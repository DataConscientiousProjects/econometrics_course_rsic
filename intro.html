Software

Install GRASS GIS and ArcGIS software following the instructions bellow.

GRASS GIS

  • Free and open source GRASS GIS: GRASS GIS 7.0
  • MS Windows 8+ (or Windows 7): use the stand-alone installer package WinGRASS 7.0.0beta4, or get the latest development version
  • Troubleshooting GRASS GIS on Windows:
    • Make sure you check install "Important Microsoft Runtime Libraries" during the installation, see screenshot1 and screenshot2.
    • If GRASS crashes at the start, you may be missing some dll files, for more details see suggested solution.
    • If you have more problems see Troubleshooting GRASS on Windows wiki page
    • You may need to install TIFF support to view images saved by the 3D visualization module
  • Mac OS X 10.7+: download and install required frameworks and GRASS7.0.0beta3 (start with the frameworks and then install GRASS-7.0beta3)
  • Linux: GRASS 7.0 packages are available in software repositories
  • use Help page to learn how to work with the software using wxGUI
  • use commands' manual pages to understand modules and their options
  • many examples are adopted and updated from the GRASSbook

ArcGIS

  • Proprietary ArcGIS Version 10.2
  • use ESRI virtual campus, available at NCSU, to learn the software
  • use help pages to understand modules and their options
