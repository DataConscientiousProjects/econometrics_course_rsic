Homework structure to be used for all assignments

  • Short paper: min. 500 word text, 1-2 pages of graphics, single spaced, no bullets - full sentences.
  • Both GRASS and ArcGIS solutions are required, described in a single paper (more details see below).
  • Step-by-step instructions are provided, but it is your task to explain what is being done in terms of methods and results, based on the lectures, manual pages and other on-line resources.
  • Assigned on Thursdays, due the following Tuesdays at 11 pm, to be submitted on google site according to the instructions on Course logistics web page.
  • Required report sections:
    1. Task, problem, background
    2. Approach: general description of methods and concepts, do not repeat instructions and do not discuss software issues here - this section should be software independent, common for both Arc and GRASS.
    3. Results: maps, graphs, tables with comments, focus on what is interesting in the data and the derived maps, how does the method used influence the results. Include answers to questions in this section. Post 4-6 images per page.
    4. Discussion: was there anything unexpected or problematic in the results? were there any issues related to the methdology? also discuss the software issues, including fixes and suggestions for improvements here
    5. Conclusion: 1-2 sentences what you have learned
  • Sample report formatting in MSWord (do not use the text from this example, it is for formatting only!). Note that the submitted report should be in pdf.

Software and data

  • Install GRASS GIS and ArcGIS software following the instructions on the Course logistics webpage
    Troubleshooting GRASS on Windows:
    • If GRASS crashes at the start, you may be missing some dll files, for more details see suggested solution
    • You may need to install tiff support to view images saved by the 3D visualization module
  • Download data
    • create folders for your GIS data in a directory where you have permissions to write, for example:
      • MSWindows - C:\Users\myname\grassdata and C:\Users\myname\arcdata
      • Mac or Linux - in your home directory myname/grassdata
    • download and unpack the following datasets: (!!!make sure that your unpacking software does not create additional directory with the same name, e.g. nc_spm_08_grass7\nc_spm_08_grass7, the instructions assume you have only one layer of nc_spm_08_grass7)
      • GRASS GIS LOCATION: nc_spm_08_grass7 into grassdata (this is the same data set as North_Carolina provided with WinGRASS7)
      • ArcGIS GEODATABSE: ncrast_geodatabase_08_31_09 into arcdata
      • ArcGIS GEODATABSE: ncshape_geodatabase_02_15_10 into arcdata
    • Data are provided at three hierarchical levels:
      • entire NC state with raster data at 500m resolution: region nc_500m
      • South-west Wake County with raster data at resolutions 30m-10m: region swwake_10m
      • small watershed in rural area with data resolutions of 1m-3m: rural_1m
      • See more at GRASSbook data description
